# ✅ Чеклист: Добавление объекта в Aura

> Используй этот чеклист при создании нового объекта мебели.

## Обязательные поля

- [ ] **Название** — понятное, на русском
- [ ] **Категория** — из списка:
  - `soft` — Мягкая мебель ✅ (публичная)
  - `living` — Гостиная 🚧 (draft)
  - `bedroom` — Спальни 🚧 (draft)
  - `kitchen` — Кухни 🚧 (draft)
- [ ] **Минимум 1 фото** — качественное, на белом/нейтральном фоне
- [ ] **Описание** — опционально, но желательно

## 3D модели (для AR)

- [ ] **GLB файл** — оптимизирован:
  - Размер < 10 MB
  - Draco compression
  - Текстуры ≤ 1024px
- [ ] **USDZ файл** — для iOS Quick Look
  - Генерируется из GLB
  - Проверить открытие на iPhone

## Статус публикации

| Статус | Значение |
|--------|----------|
| `draft` | Не виден в каталоге (по умолчанию) |
| `ready` | Виден в каталоге |
| `archived` | Скрыт, но сохранён |

## Перед публикацией

- [ ] Проверить отображение на mobile
- [ ] Проверить 3D rotation
- [ ] Протестировать AR на Android
- [ ] Протестировать AR на iOS
- [ ] Изменить статус на `ready`

## После публикации

- [ ] Проверить в каталоге
- [ ] Проверить Share-ссылку
- [ ] Сообщить команде
